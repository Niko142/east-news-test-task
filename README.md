# East News Test Task

## Описание

Визуальная утилита для анализа криптовалютного графика BTCUSD. Реализовано как одностраничное приложение с возможностью масштабирования на другие торговые пары.

## Основные особенности

- Отображение свечного графика на базе `lightweight-charts` и сгенерированных `mock`-данных
- `Mock`-данные на основе детерминированной генерации
- Переключение диапазона времени на графике: `1D`, `1W`, `1M`, `6M`, `1Y`
- Сформированы индикаторы `MACD`, `MFI` в виде графика
- Управление отображением индикаторов через боковую панель (в реализации, `UI` реализован)

> Данные синтетические и не отражают реальные рыночные показатели

## Запуск и установка

1. Клонировать репозиторий

   ```bash
   git clone https://github.com/Niko142/east-news-test-task.git
   cd east-news-test-task
   ```

2. Установить зависимости

   ```bash
   # Установка зависимостей
   npm install
   # или
   pnpm install
   ```

3. Запустить проект

   ```bash
   # Запуск в режиме разработки
   npm run dev

   # Сборка
   npm run build

   # Запуск production-сборки
   npm start
   ```

Открыть в браузере: [http://localhost:3000](http://localhost:3000)

В проекте организован автоматический редирект на `/chart/BTCUSD`.
